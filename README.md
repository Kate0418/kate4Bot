# けぃと°４号

## 概要
- ディスコードの読み上げボットです
- 一人一人に対して別々のボイスを設定可能
- 98種類のボイス搭載

## コマンド一覧
- コマンド説明
```bash
?help
```

- ボイスチャット入室
```bash
?join
```

- ボイスチャット退出
```bash
?leave
```

- ボイス変更
```bash
?set ボイスID
```

- ボイス一覧
```bash
?voice
```
